# Plan dan Hasil Eksekusi Penyesuaian CSS ke 1920x1080

## Tujuan
Mengubah CSS agar layout utama website benar-benar pas pada resolusi `1920 x 1080` pixel.

## Plan
1. Audit aturan CSS yang mengunci ukuran layout utama dan identifikasi sumber selisih terhadap target `1920x1080`.
2. Ubah ukuran fixed container utama menjadi tepat `1920px x 1080px` dengan dampak sekecil mungkin pada struktur layout.
3. Verifikasi hasil perubahan pada file yang terpengaruh dan dokumentasikan hasil eksekusinya.

## Eksekusi
Perubahan dilakukan pada [css/style.css](/Users/imanda/Code/Web/Sekolah/Jadwal-Shalat/css/style.css):

- Menambahkan aturan `html, body` menjadi `width: 100%` dan `height: 100%`.
- Mengubah `.bg` dari ukuran berbasis `em` menjadi ukuran tetap:
  - `width: 1920px`
  - `height: 1080px`
- Menambahkan `top: 0` dan `left: 0` pada `.bg` agar titik awal render konsisten di pojok kiri atas.
- Menghapus `font-size: 16px` yang sebelumnya hanya dipakai sebagai basis perhitungan `em`.

## Hasil
Sebelum perubahan, container `.bg` memiliki ukuran:

- `121em x 68em`
- Dengan basis `16px`, hasilnya `1936px x 1088px`

Setelah perubahan, container `.bg` menjadi:

- `1920px x 1080px`

## Dampak
- Layout utama sekarang tepat sama dengan target Full HD.
- Risiko tampilan terpotong di layar `1920x1080` berkurang karena tidak ada lagi selisih `16px` lebar dan `8px` tinggi.
- Struktur visual lain tetap dipertahankan karena perubahan hanya difokuskan pada root layout container.

## Catatan
Website ini masih menggunakan layout fixed-size, jadi perubahan ini membuat tampilan presisi untuk layar `1920x1080`, tetapi belum membuatnya responsif untuk ukuran layar lain.
