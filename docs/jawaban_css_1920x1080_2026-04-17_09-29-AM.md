# Jawaban Perubahan CSS ke 1920x1080

Plan untuk mengubah CSS supaya benar-benar pas `1920x1080` adalah:

1. Audit ukuran fixed pada container utama yang saat ini masih meleset dari Full HD.
2. Ganti ukuran container utama ke `1920px x 1080px` tanpa mengubah struktur layout yang sudah ada.
3. Tambahkan aturan dasar pada root element agar area render konsisten.
4. Verifikasi hasil akhir lalu dokumentasikan perubahan di folder `docs`.

Plan tersebut sudah dieksekusi.

Perubahan utamanya ada di [css/style.css](/Users/imanda/Code/Web/Sekolah/Jadwal-Shalat/css/style.css):

- Menambahkan `html, body { width: 100%; height: 100%; }`
- Mengubah `.bg` menjadi:
  - `top: 0`
  - `left: 0`
  - `width: 1920px`
  - `height: 1080px`
- Menghapus ukuran berbasis `em` yang sebelumnya membuat layout menjadi `1936x1088`

Hasil akhirnya, ukuran layout utama sekarang benar-benar `1920 x 1080` pixel.
