# Jadwal Sholat - BQ Islamic Boarding School

Aplikasi web dashboard untuk menampilkan jadwal sholat real-time yang dirancang khusus untuk kebutuhan display masjid atau pesantren. Aplikasi ini memiliki desain premium dengan animasi halus dan antarmuka yang responsif.

## 🌟 Fitur Utama

- **Real-time Clock**: Menampilkan jam digital yang akurat detik demi detik.
- **Kalender Ganda**: Menampilkan tanggal Masehi dan Hijriah secara otomatis.
- **Jadwal Sholat Otomatis**: Menghitung waktu Subuh, Dhuha, Zuhur, Ashar, Maghrib, dan Isya berdasarkan lokasi.
- **Informasi Petugas**: Slot khusus untuk menampilkan nama Muadzin dan Khatib hari ini.
- **Desain Premium**: Menggunakan tema warna Bina Qurani dengan latar belakang dinamis dan pola geometris Islam.
- **Fixed Aspect Ratio**: Dioptimalkan untuk layar display dengan rasio tetap menggunakan container `.bg`.

## 🛠️ Teknologi yang Digunakan

- **HTML5**: Struktur semantik untuk web.
- **CSS3**: Layouting menggunakan Flexbox, CSS Variables, dan kustom gradien/pola.
- **JavaScript**: Logika kalkulasi waktu sholat dan update DOM secara dinamis.
- **Font**: Poppins & Lato (Google Fonts).
- **Icons**: Font Awesome 6.

## 🚀 Cara Menjalankan Secara Lokal

1. Clone repositori ini:
   ```bash
   git clone https://github.com/imandaidfa/Jadwal-Shalat.git
   ```
2. Buka folder projek:
   ```bash
   cd Jadwal-Shalat
   ```
3. Buka file `index.html` di browser favorit Anda.

## 📂 Struktur Folder

- `index.html`: File utama aplikasi.
- `css/style.css`: File styling utama.
- `js/`: Berisi skrip logika waktu dan jadwal sholat.
- `images/`: Aset gambar dan logo.
- `docs/`: Dokumentasi tambahan dan analisis kode.
